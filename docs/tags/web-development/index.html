Building This Site

I built this website using basic HTML, CSS, JS, and the Bootstrap framework. The particles on the homepage where created using open source software particle.js. Looking back at it, using Bootstrap was completely overkill for this project since I only used for its 12 grid system. A more appropriate CSS framework would have been Skeleton, which I might migrate to in the near future. In order to create blog posts, I had to learn how to use static site generator Hugo.
