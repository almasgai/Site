Building This Site

I built this website using basic HTML, CSS, JS, and the Bootstrap framework. The particles on the homepage where created using open source software particle.js. Looking back at it, using Bootstrap was completely overkill for this project since I only used for its 12 grid system. A more appropriate CSS framework would have been Skeleton, which I might migrate to in the near future.

In order to create blog posts, I had to learn how to use static site generator Hugo. By opting for Hugo my website does not use a database. As a result, web pages load faster since data does not need to be retrieved from a database. As for the Hugo theme (which stylizes the blog portion of my site), I chose the very simple and very popular theme Kiss. For hosting, I chose to use my Github account since it’s free.
